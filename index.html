Traffic Light Controller

Technology: 555 Timer IC, CD4017 Decade Counter, 4026 IC, 7-Segment Display

Designed and developed an automatic traffic light controller for a four-way junction using digital electronic components. The system utilizes a 555 Timer as the clock generator, a CD4017 Decade Counter for signal sequencing, and a 4026 IC to drive the seven-segment countdown display. The project demonstrates efficient traffic signal management without using any microcontroller or programming.

  • Automatic Red, Yellow and Green signal operation.
  • Countdown timer using 7-segment display.
